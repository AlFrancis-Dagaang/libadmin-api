# LibAdmin API

> A Traditional Library Management System - A comprehensive REST API for managing library operations built with Java Servlets.


## Overview

LibAdmin API is a traditional library management system built with Java Servlets that provides complete CRUD operations for managing books, members, and borrowing transactions. Designed with simplicity and reliability in mind, it offers a RESTful interface for all core library operations including book cataloging, member management, and loan tracking.

## ✨ Key Features

- **Book Management** - Add, update, delete, and search books in the catalog
- **Member Management** - Register and manage library member accounts
- **Loan Transactions** - Issue, return, and track borrowed books
- **Advanced Search** - Search books by title, author, ISBN, or category
- **Transaction History** - View borrowing history and overdue items
- **Due Date Tracking** - Automatic calculation of due dates and overdue fines
- **Authentication** - Session-based authentication for librarians

## 🛠️ Tech Stack

- **Java Servlets** - Core API framework
- **JDBC** - Database connectivity
- **MySQL** - Relational database
- **Apache Tomcat** - Servlet container
- **Maven** - Build and dependency management
- **JSON** - Data interchange format
- **Log4j** - Logging framework
